# MatchMaking Solution

## Description
This repository contains the solution for the MatchMaking technical assignment.

## Requirements
- .NET 8 SDK

## How to Run the Project

1. Clone the repository:
   git clone https://github.com/your_username/matchmaking-solution.git

2. Navigate to the project folder:
   cd matchmaking-solution

3. Restore dependencies:
   dotnet restore

4. Build the project:
   dotnet build

5. Run the application:
   dotnet run
